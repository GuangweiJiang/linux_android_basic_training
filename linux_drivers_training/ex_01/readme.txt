Linux driver training material - exercise 01
by guangwei@Jan11'10

Targets:
Write a simple kernel module, make it, and insert/remove from system.

Test Enviroment:
	Ubuntu 8.04 or later. 

Time required:
	2 days

Purpose:
	1. Learn module basic conception;
	2. Learn how to write Makefile and make the kernel module (*.ko file);
	3. Learn insert module (insmod), remove module (rmmod), and list current loaded module (lsmod);
	4. Learn how to read printk debug message (cat /var/log/messages).

Refer books:
	"Linux device drivers(3rd Edition)", chapter 2 - "Building and Running Modules"
